「＊＊試験ドットコム」の過去問道場を使いやすくするためのJavaScriptです。


## 対応しているサイト
- ITパスポート試験ドットコム
- 情報セキュリティマネジメント試験ドットコム
- 基本情報技術者試験ドットコム
- 応用情報技術者試験ドットコム
- 情報処理安全確保支援士ドットコム
- ネットワークスペシャリスト ドットコム
- データベーススペシャリスト ドットコム
- プロジェクトマネジメント試験ドットコム


## 使い方
Violentmonkeyを使用することを推奨しますが、他のツールでも動作するはずです。

Violentmonkeyのダウンロードはこちら([Google Chrome][violentmonkey-chrome], [Firefox][violentmonkey-firefox])
